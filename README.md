# Gulp-Starting
Gulp starting template for web development.
<details>
  <summary>Includes packages:</summary>
  
  - gulp-sass;
  - gulp-concat;
  - browser-sync;
  - gulp-uglify-es;
  - gulp-autoprefixer;
  - gulp-imagemin;
  - del
  
</details>
<details>
  <summary>Commands:</summary>
  
 - $ gulp (for development);
 - $ gulp build (for build; production-ready files will be moved to 'dist' folder);
  
</details>
